# Crashlens Python SDK (stub)

Client SDK for reporting errors from Python applications to a Crashlens
instance. Not implemented yet; this directory is a placeholder.

Planned scope:

- One-line initialisation with a DSN.
- Flask and FastAPI integration.
- A logging handler.
- A background, non-blocking sender.
- No hard third-party dependencies.

The event payload this SDK will send is specified in
[`../../docs/PROTOCOL.md`](../../docs/PROTOCOL.md) (DRAFT).
